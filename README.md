# ClackyAI 内容运营工作目录

## 目录结构

```
clacky/
├── INPUT/                   # 投入
│   ├── _db/                 # 客户素材库（他们的资料）
│   ├── _config/             # 配置
│   │   ├── sources.yaml     # 信源清单（官网/社媒/GitHub/竞品）
│   │   └── brand.md         # 品牌调性/红线
│   ├── _sources/            # 从互联网/各平台收集的参考内容
│   └── meetings/            # 会议纪要
├── OUTPUT/                  # 产出
│   ├── Alice/              # Alice 的产出
│   ├── Clacky-CC/          # Clacky-CC 的产出
│   └── Clacky-XS/          # Clacky-XS 的产出
└── ASSETS/                  # 图片/音视频素材
```

## 运营节奏

| 类型 | 频率 | 内容 |
|------|------|------|
| 日更 | 每天 | 公众号/小红书/知乎 等 |
| 社群 | 定期 | 利他性质分享 |

## 信源配置说明

编辑 `INPUT/_config/sources.yaml` 添加/更新信源：

- **官方渠道**：官网、博客、GitHub、社媒账号
- **竞品参考**：同类产品动态
- **技术账号**：行业 KOL、技术社区
- **背书信息**：媒体报道、投资方等

## 产出命名建议

文件命名格式：`YYYY-MM-DD_平台_标题.扩展名`

示例：
```
2026-04-14_公众号_为什么L3是AI编程最优解.md
2026-04-14_小红书_5分钟上手ClackyAI.md
```
