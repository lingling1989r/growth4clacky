# Clacky 用户旅程 KG v2 — 五阶段 × 人群画像 × 内容选题下钻

> **核心逻辑**：
> - **第一优先级**：让用户**先用起来**（自用好用的第一阶段）
> - **第二优先级**：用户用好后**主动分享**（口碑传播/Skill赚钱的第二阶段）
> - 内容节奏：需求触发 → 搜索对比 → 选型评估 → 付费转化 → 口碑裂变

---

## 用户旅程总览图

```
阶段一【需求识别】
└── 痛点触发 → "我有什么问题" → 不知道自己不知道 / 知道但没解决方案

阶段二【信息搜索】
└── 主动搜索 → "有什么工具能用" → 搜索关键词 → 开始了解品类

阶段三【方案评估】
└── 对比选型 → "哪个更好" → 品牌对比 / 功能对比 / 口碑对比

阶段四【购买决策】
└── 付费转化 → "值不值" → 价格敏感 / 信任背书 / 试用决策

阶段五【购后决策】
└── 使用体验 → "好不好用" → 续费/扩量/口碑/退费 → Skill变现/推荐给他人
```
